## Requirements
Make sure the following are installed on your computer:
-   [git](https://git-scm.com/)
-   [node.js](https://nodejs.org/)
-   An IDE or Text Editor, like [Visual Studio Code](https://code.visualstudio.com/)

## Prerequisites
Some experience with basic front-end web development and JavaScript is required. The following are some helpful resources to learn:
-   [Eloquent JavaScript](https://eloquentjavascript.net/)
-   [MDN Technology for Developers Tutorials](https://developer.mozilla.org/en-US/docs/Web/Tutorials) - Introductory Level Content for HTML, CSS and JavaScript
