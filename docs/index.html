Getting Started

Requirements#

Make sure the following are installed on your computer:

  • git
  • node.js
  • An IDE or Text Editor, like Visual Studio Code

Prerequisites#

Some experience with basic front-end web development and JavaScript is required. The following are some helpful resources to learn:

  • Eloquent JavaScript
  • MDN Technology for Developers Tutorials - Introductory Level Content for HTML, CSS and JavaScript
